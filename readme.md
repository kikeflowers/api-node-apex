# api-node-apex
Api del video de apex
<br><br>
Clona el proyecto<br>
<code>git clone https://github.com/kikeflowers/api-node-apex</code>
<br><br><br>
Entra a la carpeta<br>
<code>cd api-node-apex</code>
<br><br><br>
Descarga las dependencias<br>
<code>npm install</code>
<br><br><br>
Sube el proyecto
<code>npm start </code>
